## Instalasi Proyek

```
npm install
composer install
```

```
## Fix if php error
composer self-update
composer clear-cache
rm -rf venor
rm composer.lock
composer install --ignore-platform-reqs
```

```
php artisan migrate:refresh --seed
php artisan storage:link
```
